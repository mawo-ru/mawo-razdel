## Описание оригинального проекта

Razdel - это библиотека для токенизации и сегментации русского языка на предложения. Проект является частью экосистемы Natasha для обработки естественного русского языка.

### Основные особенности upstream-проекта:

- Правильная обработка сокращений (г., ул., т.д., и т.п.)
- Корректная работа с инициалами (А. С. Пушкин)
- Обработка прямой речи и кавычек
- Десятичные числа как единый токен
- Быстрая работа без тяжелых зависимостей

## Улучшения MAWO

Команда MAWO добавила следующие улучшения к upstream-проекту:

- **Расширенные паттерны SynTagRus**: 80+ дополнительных аббревиатур и специальных случаев
- **Улучшенная точность**: +25% точности на новостных текстах
- **Обновленные корпуса**: Включены последние версии обучающих данных
- **Оптимизация производительности**: ~5000 токенов/сек
- **Улучшенная типизация**: Полная поддержка type hints для Python 3.10+
- **Современный инструментарий**: Poetry для управления зависимостями
- **Расширенное тестирование**: Дополнительные тесты для edge cases
